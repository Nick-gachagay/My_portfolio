Gachaga OS

Gachaga OS

Custom Linux-based OS built from Qubes OS principles, focusing on security, virtual isolation, and penetration testing tools. Key features:

  • Xen-based virtualization for compartmentalization
  • Pre-configured security tools (Metasploit, Wireshark, Nmap)
  • Secure inter-VM communication channels
  • Custom kernel hardening patches
# Sample build command
$ qubes-builder --setup --config builder-gachaga.conf

Python Personal Database

Minimalist local database built with Python for storing personal records and activity logs securely. Features include:
